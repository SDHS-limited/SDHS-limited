### [Vtuber Desktop](https://github.com/SDHS-limited/Vtuber-Desktop) — AI VTuber Desktop App
Unity + Live2D로 구현한 AI 버튜버 데스크탑 앱. 스페이스바를 누른 채 말하면 Whisper가 음성을 인식하고, 로컬 LLM이 답변을 생성해 VOICEVOX로 캐릭터가 직접 말한다. 말하는 동안 Live2D 입 파라미터가 오디오 볼륨에 연동되어 자연스러운 립싱크가 구현된다.
Whisper STT → Ollama LLM → VOICEVOX TTS 파이프라인 · Live2D 오디오 립싱크 · 캐릭터 드래그 이동
 
`Unity` `C#` `Live2D` `Whisper` `Ollama` `VOICEVOX` `Python`

### [NutriAI](https://github.com/SDHS-limited/NutriAi) — AI Nutrition App
로컬 LLM(Ollama)을 활용한 AI 개인 영양사 웹앱. 음식 사진을 찍으면 영양소를 분석하고, AI와 식단 상담이 가능.
Ollama 로컬 LLM 연동 · Vision 모델 음식 이미지 인식 · 주간 영양 인사이트
 
`React` `Node.js` `Express` `Ollama` `Tailwind CSS`
